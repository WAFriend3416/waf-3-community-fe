개발 포트폴리오

어떻게 만들었나요?

⚙️

Backend

Spring Boot 3.5 + Java 24로 REST API 구현. JWT 인증, MySQL 데이터베이스, AWS S3 이미지 저장소 사용

🎨

Frontend

Vanilla JavaScript + Express.js로 빌드 없이 심플하게. BEM 명명 규칙과 컴포넌트 기반 CSS 아키텍처 적용

🔐

Security

HttpOnly Cookie로 XSS 방지, CSRF 토큰, Rate Limiting으로 안전한 인증 시스템 구축

⚡

Performance

Cursor 기반 무한 스크롤, N+1 쿼리 최적화, 원자적 UPDATE로 동시성 제어 구현
